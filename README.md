# Suma con Árbol de Decisiones

Este es un pequeño proyecto para aprender a sumar dos números utilizando un Árbol de Decisiones (Decision Tree) con la biblioteca scikit-learn en Python.

## Descripción

El objetivo de este proyecto es entrenar un modelo de Árbol de Decisiones para que pueda predecir la suma de dos números. Se generan 10,000 ejemplos de entrenamiento con pares de números aleatorios entre 1 y 100. Luego, el modelo se entrena con estos datos para aprender a sumar.

## Requisitos

- Python 3
- scikit-learn
